# Simple Counter 

It's just a simple number counter, that's all.

## Features

1. Increase, reset and decrease buttons
2. Hover button effect

## Tech Stack
- HTML5
- CSS3
- JavaScript

## How to run
1. Download or clone this repository
2. Navigate to the ``number-counter`` folder
3. Open ``main.html`` in your web browser
4. Click the buttons
---
Or, press this link => [DEMO](https://marvelttihu.github.io/number-counter/)

## Notes

This is intended as a learning project for Javascript basics
